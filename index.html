Step-by-Step

Installing the NAS OS

Two paths: OpenMediaVault (recommended for Pi and laptops) or TrueNAS SCALE (x86 with 8GB+ RAM).

1

Flash Raspberry Pi OS Lite

Open Raspberry Pi Imager ‚Üí Choose Pi OS Lite (64-bit) ‚Üí Set hostname, enable SSH, set password ‚Üí Write to SD card.

2

Boot & Connect via SSH

Insert SD card, connect Ethernet + power. Wait 90 seconds, then:

ssh pi@piNAS.local
3

Update the System

sudo apt update && sudo apt full-upgrade -y && sudo reboot
4

Install OpenMediaVault

wget -O - https://github.com/OpenMediaVault-Plugin-Developers/installScript/raw/master/install | sudo bash

Takes 15‚Äì30 minutes. Do NOT interrupt the process.
5

Access Web UI

Default login: admin / openmediavault ‚Äî change password immediately.

6

Mount Your Storage Drive

Navigate to Storage ‚Üí Disks (verify drive detected) ‚Üí File Systems ‚Üí Create (EXT4) ‚Üí Mount ‚Üí Apply.
